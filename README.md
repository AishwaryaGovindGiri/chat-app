Chat App

Real-time chat application built using MERN Stack and Socket.IO.

Features:
- Authentication (JWT)
- Real-time messaging
- Online users
- Profile management

Tech Stack:
- React
- Node.js
- Express.js
- MongoDB
- Socket.IO
